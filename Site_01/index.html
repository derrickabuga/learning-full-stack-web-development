Heading Elements

The heading elements ( h1 to h6 ), are used to add headings to web pages; the higher the number, the lower the importance.

Code:

Heading one: ..........

<h1></h1>
Heading two: ..........
<h2></h2>
Heading three: ........
<h3></h3>
Heading four: .........
<h4></h1>
Heading five: .........
<h5></h5>
Heading six: ..........
<h6></h6>
